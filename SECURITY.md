# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Astra CMS, please report it responsibly.

**Do not open a public issue.** Instead, please use [GitHub's private vulnerability reporting](https://github.com/Kaimaan/astra-cms/security/advisories/new) to submit your report.

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation plan within 7 days for critical issues.

## Scope

This policy applies to the Astra CMS codebase at [github.com/Kaimaan/astra-cms](https://github.com/Kaimaan/astra-cms).
